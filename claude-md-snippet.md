## Дискавери скиллов (memoskill)

В Claude Code в контекст инжектится лишь **подмножество** установленных скиллов, и видно только имя (без `description`) — поэтому релевантный скилл легко пропустить. Полный список лежит в `~/.claude/SKILLS-INDEX.md` (плоский `name :: description` по всем ~1700 скиллам, включая плагинные).

**Перед началом доменной задачи** (frontend / UI / motion / data / security / devops / docs / маркетинг / research / …), ДО написания кода:

1. Грепни индекс по 2–4 ключевым словам домена:
   `grep -iE "ключslovo1|ключslovo2" ~/.claude/SKILLS-INDEX.md`
2. Просмотри **все** совпадения — не якорись на первом. Если кандидатов несколько (напр. `taste` vs `impeccable`), сравни по `description` и выбери самый специфичный под задачу.
3. Нашёл релевантный — вызови его через `Skill` до начала работы.

Регенерация индекса (после установки/удаления скиллов или плагинов):
`npx skillseek index`
